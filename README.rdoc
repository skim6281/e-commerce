= E-Commerce

== Usage
- Setup
  - git clone https://github.com/skim6281/e-commerce.git
  - bundle install
  - npm install
  - webpack
  - bundle exec rails s

- Go to http://localhost:3000

== Technologies
* Ruby on Rails v4.2.8
* React v15.6.1
* Redux
* PostgreSQL
* SASS
* Gems
  * BCrypt
  * Jbuilder
  * Figaro
